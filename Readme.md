## ⚡ Active Projects & Research

### 🤖 [J.A.R.V.I.S AI Assistant (Mark XL)](https://github.com/durd1ma7ov06/Jarvis)
> **Offline-first AI Assistant inspired by Tony Stark's JARVIS.**
> - **Speech-to-Text**: High-accuracy `faster-whisper` offline voice recognition.
> - **LLM Brain**: Local Ollama (Qwen2.5:7b) + Cloud OpenRouter.
> - **Text-to-Speech**: Crystal clear natural speech via EdgeTTS.
> - **HUD Console**: Futuristic PyQt6 dashboard with rotating hologram vectors and CPU metrics.

### 🎭 Humanoid-ALI Robot Core
> **Interactive hardware & software platform for custom humanoid robots.**
> - Integrating PCA9685 servo drivers for realistic human-like arm and neck gestures.
> - Real-time 3D mouth deformation synced directly with sound wave amplitudes.
> - Multilingual support (Uzbek, Russian, English) with background wake-word triggers.
