## ⚡ Active Projects & Research

### 🤖 [J.A.R.V.I.S AI Assistant (Mark XL)](https://github.com/durd1ma7ov06/Jarvis)
> **Fully offline-first AI Assistant inspired by Tony Stark's JARVIS, featuring local system control.**
> - **Speech-to-Text**: Ultra-low latency voice transcription using localized `faster-whisper` models.
> - **LLM Brain**: Adaptive local reasoning powered by Ollama (Qwen2.5:7b) with seamless transition to OpenRouter API (Gemini/GPT) when required.
> - **Text-to-Speech**: Neural voice synthesis engine integrating Microsoft EdgeTTS for realistic Uzbek voice flow.
> - **Holo Dashboard**: Cyberpunk HUD design written in PyQt6 with complex QPainter rotation diagnostics, system hardware monitoring, and interactive sound waves.
> - **System Automation**: Capable of executing shell scripts, opening applications, scanning system metrics, running browser automation scripts, and searching the web.

### 🎭 [Humanoid-ALI Robot Core](https://github.com/durd1ma7ov06/Humanoid-ALI--master)
> **Complex humanoid robot brain architecture syncing real-time voice synthesis with servo-motor actions.**
> - **Hardware Sync**: Interfaces Python with PCA9685 I2C servo drivers via ESP32, translating text emotions to precise arm and neck movements.
> - **Lip Sync**: Procedural lip-sync rendering engine dynamically deforming a 3D mouth vertex mesh depending on real-time speech amplitudes.
> - **RAG Integration**: Custom localized Retrieval-Augmented Generation indexing historical databases to keep the robot's context historically accurate.
